# DefinedIM

不受限制的即时通讯

*本仓库为服务端仓库*

## Features
+ 基于 `SM2` 非对称加密算法的端到端加密通讯
+ 多平台客户端（计划中）
+ 去中心化的设计理念，适于中小型组织内部使用
+ 高度可配置的服务端
+ 支持插件自由扩展


## 构建

### 依赖

JDK 17  
Rust ~~nightly~~ 任意通道最新工具链  
Gradle
